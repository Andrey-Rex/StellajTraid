Правила обработки персональных данных

Настоящие Правила обработки персональных данных описывают, как и для каких целей мы собираем, используем и защищаем ваши персональные данные при использовании нашего сайта.

1. Сбор персональных данных

Мы можем собирать следующие данные:

  • ФИО, контактные данные (телефон, email)
  • Информацию о взаимодействии с сайтом (IP-адрес, время посещения, cookies)
  • Другие данные, которые вы добровольно предоставляете через формы обратной связи

2. Цели обработки данных

Персональные данные обрабатываются для следующих целей:

  • Обеспечение работоспособности сайта и улучшение сервиса
  • Связь с пользователями по вопросам, связанным с использованием сайта
  • Выполнение юридических обязательств

3. Правовые основания обработки

Обработка персональных данных осуществляется на основании вашего согласия, а также в случаях, предусмотренных законодательством.

4. Защита данных

Мы принимаем меры для защиты персональных данных от несанкционированного доступа, изменения и распространения.

5. Передача данных третьим лицам

Ваши данные не передаются третьим лицам без вашего согласия, за исключением случаев, предусмотренных законом.

6. Права пользователей

Вы имеете право:

  • Получать информацию об обработке ваших данных
  • Исправлять или удалять свои данные
  • Отзывать согласие на обработку данных

7. Контактная информация

По вопросам, связанным с обработкой персональных данных, вы можете связаться с нами по электронной почте: privacy@yourwebsite.com
